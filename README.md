# SDK installation

## SPM

Add our SDK as a dependency to your project, like:

```swift
.package(url: "https://github.com/solutionarchitectstech/mobile_sdk_release_ios", exact: "1.7.0")
```

## Cocoapods

```
pod 'SAAdvertisingSDKStandard', :git => 'https://github.com/solutionarchitectstech/mobile_sdk_release_ios.git', :tag => '1.7.0'
```
